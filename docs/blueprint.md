# **App Name**: Quiz Whiz

## Core Features:

- Question Display: Display a question with multiple choice answers.
- Answer Selection: Allow users to select an answer.
- Feedback: Provide immediate feedback on whether the answer was correct or incorrect.
- Score Tracking: Tally the score to show how well the quiz taker performed.
- Topic Recommendation: Based on quiz results, suggest the next quiz topic.
- Question Import: Import questions from a text or JSON file.

## Style Guidelines:

- Primary color: Soft, muted rose (#F08080) to create a calm and focused environment.
- Background color: Light grayish-pink (#F8E8E8), creating a subtle contrast with the primary color.
- Accent color: Muted orange (#E59766) to highlight important elements like the correct/incorrect feedback.
- Headline font: 'Belleza', a humanist sans-serif for titles and important text, which creates a friendly and inviting feel; body font: 'Alegreya', a humanist serif for displaying the question/answer text to maximize readability.
- Simple, clear icons to represent different quiz categories (e.g., history, science, literature).
- Clean and organized layout with clear visual hierarchy to guide the user through the quiz.
- Subtle animations for transitions between questions and when revealing correct/incorrect answers.
